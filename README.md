# LED 数码管依次显示 by Arduino UNO

## 调试过程
先用 B11111110 调(从高到低，D7~D0)，1代表不亮，0代表亮，然后用D0去试，D0接哪哪亮，然后画图。
画出下面这个图，然后按顺序插线，从D0插h 到 D7插d，然后要显示什么数字就相应调整。
